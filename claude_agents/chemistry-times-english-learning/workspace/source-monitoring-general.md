## Usage Notes

### For Digital Journalist
- Check wire services first for breaking stories
- Use global media for analysis and context
- Cross-reference Asia-Pacific sources for Taiwan relevance angle
- Science/health sources are checked when editorial brief includes those topics

### For World News Analyst
- Wire services provide the factual baseline
- Global media provides analytical frameworks
- Asia-Pacific sources provide regional context
- Always connect stories back to tech industry or Taiwan relevance

### Tier Usage Rules

| Tier | Definition | Usage Rule |
|------|-----------|------------|
| Tier 1 | Official sources, government, peer-reviewed | Full use — highest credibility |
| Tier 2 | Established media with editorial staff | Full use — standard credibility |
| Tier 3 | Social media, forums, blogs | Must corroborate with Tier 1/2 |

### Maintenance

Review monthly. Add sources demonstrating consistent quality over 3+ months. Remove sources that cease publication or shift focus.
